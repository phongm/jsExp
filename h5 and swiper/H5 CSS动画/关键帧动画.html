关键帧动画

animation(-o-, -moz-, -webkit-, -ms-)

animation{name, duration, timing-function, delay, iteration-count, direction}

name : 动画定义的名字

duration : 动画需要的时间

timing-function : 时间曲线（step-start， step-end）

delay : 延迟时间

iteration-count : number | infinite

direction : 是否允许反播 alternate
