1、html5：Document root element.<br>文档根元素
2、col:Columns in a table.<br>表格中的列
3、table：Table of multi-dimensional data.<br>表格
4、head：First element of the HTML document. Contains document metadata.<br>HTML文档中的第一个元素。包含文档元数据
5、span：Container with no semantic meaning.<br>无语义的容器
6、div：Container with no semantic meaning.<br>无语义的容器

8、form：Form.<br>表单
9、body：Document content.<br>文档的主体
10、h1：Heading for the current section.<br>当前章节的标题
11、section：Contains elements grouped by theme, for example a chapter or tab box.<br>包含同主题一组元素的容器，例如文档的章节
12、colgroup：Defines a group of columns in a table.<br>对表格中的若干列进行组合
13、tr：A row of cells.<br>表格中的一行单元

18、select：Control for selecting from multiple options.<br>有多选值的下拉列表
19、aside：Content related to surrounding elements that doesn't belong inline, such as a advertising or quotes.<br>与周围元素相关的内容，但分开显示
20、h2：Heading for the current section.<br>当前章节的标题
21、header：Navigation or introductory elements for the current section.<br>当前章节的导航或引导性元素

27、em：Text that should be emphasised.<br>需要强调的文本
28、i：Text in a alternate voice or mood, such as a technical term.<br>斜体的文本
29、small：An aside, such as fine print.<br>更小的旁注信息
30、ins：Text that has been inserted during document editing.<br>在文档编辑过程中插入的文本

32、br：Line break.<br>换行符
33、p：Paragraph content.<br>段落内容

44、abbr：Abbreviation or acronym.<br>缩写或缩略词
45、time：Time defined in a machine readable format.<br>时间
46、b：Stylistically separated text of equal importance, such as a product name.<br>粗体文本，用于强调
47、strong：Text that is important.<br>重要的文本
48、del：Text that has been removed during document editing.<br>在编辑过程中被删除的文本
49、kbd：Example input (usually keyboard) for a program.<br>由键盘输入的文本
50、hr：Thematic break within a paragraph.<br>水平线/内容主题的变化

53、label：Caption for a form control.<br>表单控件的标题（标签）
