meta 标签 辅助性标签 帮助浏览器更好的了解执行该网页

http-equiv | name

http-equiv: 它跟http请求很像 它主要帮助浏览器执行，获取一些状态

name: 它主要是帮助浏览器更好的显示 获取信息，方便seo优化

viewport: 设备观点

device-width | height: 设备宽度和高度

width | height : 渲染的宽度和高度

initial-scale : 初始化的放大缩小（0 ~ 10）

maximum-scale : 最大的放大倍数（0 ~ 10）

minimum-scale : 最小的放大倍数（0 ~ 10）

user-scalable : 是否允许用户放大缩小

--------------------------------------------------------------------

apple-mobile-web-app-capable : 全屏显示

通常和apple-mobile-web-app-status-bar-style : 状态栏样式 一旦定义为black 就会有一个黑色的背景
